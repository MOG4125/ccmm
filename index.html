Mod Info

Basic metadata for your mod. This appears in Cookie Clicker's mod manager.

Cursor Multipliers

Multiply base cursor CPS or the bonus cursors receive from each building.

Building Multipliers

Set a CPS multiplier per building type. 1 = unchanged, 2 = double, 0.5 = half.

Custom Upgrades

Define new upgrades to add to the shop. They can boost buildings, clicks, or CPS globally.

Golden Cookies

Tune how often golden cookies appear and how long they last.

Blabs

Blabs are rare one-liner messages that flash on screen after a golden cookie is clicked. They do nothing — they're pure flavour. Add your own to the pool here.

Misc / Hooks

Global tweaks and game hooks to apply at mod load time.

Info.txt Preview

Preview the info.txt file content that will be generated when you export your mod. This file contains the mod metadata for Cookie Clicker's mod manager.

Bookmarklet Code

Generate a JavaScript bookmarklet you can paste into your browser's bookmarks bar to run your mod on the web version of Cookie Clicker. Click the bookmark while on the game page to apply your mod instantly — no Steam mod manager needed!

  1. Click "Refresh Code" to generate the bookmarklet from your current settings
  2. Copy the generated code
  3. In your browser, add a new bookmark (Ctrl+D or Cmd+D)
  4. Paste the code into the URL/Location field
  5. Name it something memorable (e.g. "Cookie Clicker Mod")
  6. Go to orteil.dashnet.org/cookieclicker
  7. Click your bookmark to apply the mod!
